# paykeeper



## Project setup

```bash
pnpm install
```

### Compiles and hot-reloads for development

```bash
pnpm run dev
```

### Compiles and minifies for production

```bash
pnpm run build
```

## Usage

```bash
git clone https://github.com/luv20100918/paykeeper
cd paykeeper
pnpm i
pnpm run dev
```
